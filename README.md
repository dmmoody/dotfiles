# Common Vim Plugins

## Yggdroot/indentLine
http://vimawesome.com/plugin/indentline
A vim plugin to display the indentation levels with thin vertical lines.
vim

## airblade/vim-gitgutter
http://vimawesome.com/plugin/vim-gitgutter
